핵심 특징

AI/ML 기반 종목 선정

매일 업데이트되는 머신러닝 모델이 수천 개의 종목 중 가장 승률 높은 종목을 정밀 타격합니다. 기술적 지표와 퀀트 데이터를 결합한 최적의 리스트를 제공합니다.

완전 자동 매매 시스템

장 시작 전 전략 수립부터 장중 실시간 매수/매도까지, 한 번의 클릭으로 모든 과정이 자동화됩니다. 감정의 개입 없는 철저한 원칙 매매를 실현합니다.

실시간 데이터 모니터링

키움 API 기반의 실시간 시세와 계좌 정보를 직관적인 대시보드로 한눈에 파악하세요. 현재 자산 상태와 체결 현황을 지연 없이 확인 가능합니다.

기술적 강점

안정적인 멀티 프로세스

GUI와 실행 엔진을 분리하여 시스템 중단 없는 최상의 안정성을 제공합니다. 리소스 최적화를 통해 24시간 안정적인 구동이 가능합니다.

실시간 클라우드 동기화

Supabase 클라우드를 통해 토큰 세션과 데이터를 실시간으로 관리합니다. 어디서든 동일한 환경에서 매매를 이어갈 수 있습니다.

정밀한 리스크 관리

서킷 브레이커 감지 및 자동 손절 로직으로 급변하는 시장에서도 당신의 자산을 보호합니다. 위험 요소를 사전에 차단하는 AI 방어 기제.

설치 및 키움 API 등록 방법

01

프로그램 다운로드 및 설치

상단의 '시작하기' 버튼을 클릭하여 설치 파일(`ATS_Setup.exe`)을 다운로드하고 실행합니다. 안내에 따라 설치를 완료해 주세요.

02

키움증권 Open API+ 신청

키움증권 홈페이지의 [홈페이지 > 고객서비스 > 다운로드 > Open API > 키움 Open API+] 메뉴에서 서비스 사용 신청을 완료해야 합니다.

03

Open API+ 모듈 설치

Open API+ 신청 페이지에서 제공하는 'Open API+ 모듈'을 설치합니다. 정상적인 작동을 위해 반드시 설치가 필요합니다.

04

ATS 실행 및 로그인

설치된 ATS 프로그램을 실행하고 키움증권 계정으로 로그인합니다. 최초 실행 시 API 인증 절차가 진행될 수 있습니다.

05

Open API+ 비밀번호 저장

트레이딩 바의 [기능 > 계좌비밀번호 저장] 메뉴에서 계좌 비밀번호를 입력하고 '전체계좌 저장'을 체크한 후 등록합니다. 이 과정을 거쳐야 자동 매매가 원활히 진행됩니다.

ATS 사용방법

📊

실시간 트레이딩

조건 검색을 통해 종목을 포착하고, ML 전략 버튼으로 예측 데이터를 불러와 자동매매를 시작할 수 있습니다. 스케줄러를 통해 특정 시간 예약 매매도 가능합니다.

📈

포트폴리오 관리

보유 종목의 수익률과 상태를 실시간으로 확인하세요. 종목별 우클릭 메뉴를 통해 개별 익절/손절가 및 목표 수익률을 커스텀 설정할 수 있습니다.

🔍

데이터 분석 및 조회

과거 매매 기록, 일별 수익률, 계좌 요약 정보를 상세히 조회할 수 있습니다. 집계 기능을 통해 승률과 PF 등 통계 지표를 분석해 보세요.

⚙️

시스템 설정

시장 상황 필터, Discord 알림, 트레일링 스탑 등 모든 매매 로직을 관리합니다. 설정 변경 후에는 반드시 [설정 적용] 버튼을 눌러주세요.

💡 사용 팁 및 주의사항

  • 매일 아침 최신 파일 다운로드 버튼을 눌러 최신 ML 예측 데이터를 업데이트하세요.
  • 예상치 못한 상황 발생 시 즉시 🚨 비상 정지 버튼을 활용하여 시스템을 제어하세요.
  • 자동매매 시작 전 설정 탭에서 모든 값이 올바르게 입력되었는지 확인하는 습관이 중요합니다.
