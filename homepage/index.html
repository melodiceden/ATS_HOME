지금 시작하기

핵심 특징

AI/ML 기반 종목 선정

매일 업데이트되는 머신러닝 모델이 수천 개의 종목 중 가장 승률 높은 종목을 정밀 타격합니다. 기술적 지표와 퀀트 데이터를 결합한 최적의 리스트를 제공합니다.

완전 자동 매매 시스템

장 시작 전 전략 수립부터 장중 실시간 매수/매도까지, 한 번의 클릭으로 모든 과정이 자동화됩니다. 감정의 개입 없는 철저한 원칙 매매를 실현합니다.

실시간 데이터 모니터링

키움 API 기반의 실시간 시세와 계좌 정보를 직관적인 대시보드로 한눈에 파악하세요. 현재 자산 상태와 체결 현황을 지연 없이 확인 가능합니다.

기술적 강점

안정적인 멀티 프로세스

GUI와 실행 엔진을 분리하여 시스템 중단 없는 최상의 안정성을 제공합니다. 리소스 최적화를 통해 24시간 안정적인 구동이 가능합니다.

실시간 클라우드 동기화

Supabase 클라우드를 통해 토큰 세션과 데이터를 실시간으로 관리합니다. 어디서든 동일한 환경에서 매매를 이어갈 수 있습니다.

정밀한 리스크 관리

서킷 브레이커 감지 및 자동 손절 로직으로 급변하는 시장에서도 당신의 자산을 보호합니다. 위험 요소를 사전에 차단하는 AI 방어 기제.

설치 및 키움 API 등록 방법

01

프로그램 다운로드 및 설치

상단의 '시작하기' 버튼을 클릭하여 설치 파일(`ATS_Setup.exe`)을 다운로드하고 실행합니다. 안내에 따라 설치를 완료해 주세요.

02

키움증권 Open API+ 신청

키움증권 홈페이지의 [홈페이지 > 고객서비스 > 다운로드 > Open API > 키움 Open API+] 메뉴에서 서비스 사용 신청을 완료해야 합니다.

03

Open API+ 모듈 설치

Open API+ 신청 페이지에서 제공하는 'Open API+ 모듈'을 설치합니다. 정상적인 작동을 위해 반드시 설치가 필요합니다.

04

ATS 실행 및 로그인

설치된 ATS 프로그램을 실행하고 키움증권 계정으로 로그인합니다. 최초 실행 시 API 인증 절차가 진행될 수 있습니다.

05

Open API+ 비밀번호 저장

트레이딩 바의 [기능 > 계좌비밀번호 저장] 메뉴에서 계좌 비밀번호를 입력하고 '전체계좌 저장'을 체크한 후 등록합니다. 이 과정을 거쳐야 자동 매매가 원활히 진행됩니다.

ATS 사용방법

Overview

01. 메인 화면 구성

ATS GUI는 상단 상태바, 중앙 탭 영역, 하단 로그 창으로 구성되어 있습니다. 실시간 시장 지수와 프로그램의 동작 상태를 한눈에 파악할 수 있습니다.

  • 시장 상태: 장중, 장 마감, 휴장일 정보 표시
  • 지수 정보: KOSPI/KOSDAQ 실시간 등락률
  • 로그 창: 모든 API 요청 및 매매 내역 실시간 출력
Trading

02. 실시간 트레이딩 제어

조건 검색식 선택과 머신러닝 전략 실행을 통해 자동매매를 제어합니다. 비상 상황 시 즉시 모든 동작을 중단할 수 있는 안전 장치가 마련되어 있습니다.

  • 조건 검색: 캐치(KATCH) 조건식 실시간 포착
  • ML 전략: 서버에서 최신 예측 데이터 다운로드 및 적용
  • 비상 정지: 긴급 상황 시 즉시 시스템 제어
Portfolio

03. 포트폴리오 및 개별 설정

현재 보유 중인 종목의 수익률을 실시간으로 모니터링하고, 각 종목별로 최적화된 매도 전략을 개별적으로 설정할 수 있습니다.

  • 실시간 수익률: 보유 종목별 상세 손익 현황
  • 개별 목표 설정: 우클릭 메뉴를 통한 익절/손절가 커스텀
  • 전략 구분: 각 종목이 어떤 전략으로 매수되었는지 표시
Analysis

04. 데이터 조회 및 성과 분석

과거 매매 기록과 계좌의 자산 변화를 상세히 분석합니다. 통계 지표를 통해 자신의 매매 전략을 지속적으로 개선할 수 있습니다.

  • 계좌 요약: 추정자산 및 총 평가 손익 확인
  • 매매 기록 집계: 종목별/일별 승률 및 PF 분석
  • 상세 로그: 과거 조건 포착 및 매매 내역 조회
Settings

05. 정밀한 시스템 설정

매매 임계값, 리스크 관리 로직, 알림 설정 등 시스템의 핵심 파라미터를 사용자의 투자 성향에 맞게 조정할 수 있습니다.

  • 시장 필터: 지수 급락 시 자동 매수 제한 설정
  • 알림 설정: Discord를 통한 실시간 매매 알림
  • 전략 파라미터: 트레일링 스탑 및 타임컷 상세 설정
