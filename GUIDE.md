# ATS (Automated Trading System) 가이드

이 문서는 ATS 프로젝트의 변경 사항을 깃허브에 저장하는 방법과 프로그램의 기본적인 사용 방법을 안내합니다.

---

## 1. 깃허브 저장 방법 (Git 사용법)

프로젝트의 소스 코드를 수정하거나 새로운 파일을 추가한 후, 깃허브 원격 저장소에 반영하는 절차입니다.

### 기본 명령어 순서
1. **변경 사항 확인**
   ```powershell
   git status
   ```
2. **변경 파일 스테이징** (모든 변경 사항을 추가할 때)
   ```powershell
   git add .
   ```
3. **커밋 생성** (변경 내용에 대한 설명 작성)
   ```powershell
   git commit -m "변경 내용에 대한 설명"
   ```
4. **원격 저장소로 푸시**
   ```powershell
   git push origin main
   ```

> [!NOTE]
> PowerShell에서 `&&` 연산자가 작동하지 않을 경우, 위 명령어들을 한 줄씩 차례대로 실행해 주세요.

---

## 2. 프로그램 사용 방법 (설치 및 설정)

ATS를 정상적으로 사용하기 위한 단계별 가이드입니다.

### Step 01: 프로그램 설치
- `ATS_Setup.exe` 파일을 실행하여 설치를 완료합니다.

### Step 02: 키움증권 Open API+ 신청
- [키움증권 홈페이지](https://www.kiwoom.com) 접속
- [전체메뉴 > 트레이딩 > Open API > Open API+ 소개]에서 서비스 신청

### Step 03: Open API+ 모듈 설치
- 신청 페이지에서 제공하는 'Open API+ 모듈'을 반드시 설치해야 합니다.

### Step 04: ATS 실행 및 로그인
- 설치된 ATS 프로그램을 실행하고 키움증권 계정으로 로그인합니다.

### Step 05: 계좌 비밀번호 저장 (필수)
- 로그인 후 하단 트레이딩 바의 **[기능 > 계좌비밀번호 저장]** 메뉴 클릭
- 계좌 비밀번호 입력 후 **'전체계좌 저장'** 체크 및 등록
- 이 설정을 완료해야 자동 매매 시 비밀번호 입력 창이 뜨지 않고 원활하게 작동합니다.

---

## 3. 주요 기능 안내
- **AI/ML 종목 선정**: 머신러닝 모델 기반의 승률 높은 종목 자동 추출
- **완전 자동 매매**: 장 시작부터 종료까지 원칙에 기반한 자동 매수/매도
- **실시간 모니터링**: 대시보드를 통한 실시간 자산 및 체결 현황 파악
